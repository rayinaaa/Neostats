# Retail Sales Analytics Dashboard

## Project Overview

This project focuses on analyzing retail sales data using Python and Power BI. The objective was to build an end-to-end data pipeline that performs data ingestion, cleaning, transformation, and analysis to generate meaningful business insights.

## Tools Used

* Python (Pandas)
* Jupyter Notebook
* Power BI
* Microsoft Excel

## Project Workflow

1. Loaded retail transaction and product datasets.
2. Cleaned and validated the data by handling missing values, duplicates, and inconsistencies.
3. Transformed and merged datasets to create a unified analytical dataset.
4. Calculated key business metrics and KPIs.
5. Built interactive Power BI dashboards for business analysis.

## Dashboard Pages

* Executive Dashboard
* Revenue Analysis Dashboard
* Product Performance Dashboard
* Category Insights Dashboard

## Key Outcomes

* Identified Electronics as the top-performing category.
* Analyzed revenue trends across different periods.
* Evaluated online versus offline sales performance.
* Identified top-performing cities and products.
* Generated actionable insights to support business decision-making.

## Deliverables

* Python Notebook
* Cleaned Dataset
* Power BI Report (.pbix)
* Dashboard Screenshots
* Project Documentation

This project demonstrates the complete process of transforming raw retail data into meaningful insights through data engineering, analytics, and visualization.
